- 应用数学知识少（近乎为零）
- 效果好
- 可以解释机器学习算法使用过程中的很多细节问题
- 更完整的刻画机器学习应用的流程

#### 超参数和模型参数
- 超参数：在算法运行前需要决定的参数
- 模型算法：算法过程中学习的参数

kNN 算法没有模型参数

kNN 算法中的 k 是典型的超参数
