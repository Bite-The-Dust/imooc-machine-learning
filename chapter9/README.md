# 逻辑回归 Logistic Regression

逻辑回归：解决分类问题

回归问题怎么解决分类问题：将样本的特征和样本发生的概率联系起来，概率是一个数

![逻辑回归](images/逻辑回归.png)

逻辑回归既可以看作是回归算法，也可以看作是分类算法，通常作为分类算法用，只可以解决二分类问题
